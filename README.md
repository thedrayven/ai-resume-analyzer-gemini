# AI Resume Analyzer

An AI-powered Resume Analyzer built with **Python**, **Gradio**, and **Google Gemini 1.5 Flash**.
Upload your resume PDF and get instant, professional-grade feedback — including an ATS score, strengths, weaknesses, missing skills, and tailored improvement suggestions.

## ✨ Features

- 📤 Upload Resume as PDF
- 🤖 AI Analysis using **Google Gemini 1.5 Flash**
- 📊 ATS Score out of 100
- ✅ Strengths & ⚠️ Weaknesses Detection
- 🧩 Missing Skills Identification
- 💡 Personalized Improvement Suggestions
- 🎯 Best Suitable Job Role Recommendations
- 🌑 Modern Dark-Themed UI built with Gradio
- ☁️ Runs entirely on **Google Colab** — no setup hassle

---

## 🛠️ Tech Stack

- **Python 3**
- **Gradio** – Interactive web UI
- **Google Generative AI (Gemini 1.5 Flash)** – AI analysis engine
- **PyPDF2** – PDF text extraction

---

## 🚀 How to Run

1. Open the notebook in **Google Colab**.
2. Get a free Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey).
3. Run all cells.
4. Enter your API key when prompted.
5. Open the public Gradio link and upload your resume!
